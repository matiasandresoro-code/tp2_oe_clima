

# tp2_oe_clima

Trabajo Practico N. 2. Gestion Colaborativa, Control de Versiones y Organizacion Empresarial.

Alumno: Matias Andres Oro.

Se utiliza un dataset del Servicio Meteorologico Nacional para obtener valores maximas y minimos de temperatura durante
los ultimos 365 dias. Se va a trabajar sobre los valores arrojados en el sensor "ROSARIO AERO",
que es la estacion meteorologica emplazada en el aeropuerto de la ciudad de Rosario.

Se trabaja el dataset, que viene en texto plano, mediante libreria PANDAS para su limpieza y formato,
utilizando MATPLOTLIB para la realizacion de graficos.

El script se encuentra dentro de la carpeta scripts, y se llama analisis_datos.py
Dentro de la carpeta datos se encuentra el dataset, y dentro de la carpeta resultados se encuentra el grafico del resultado.

Analisis de temperatura de la ciudad de Rosario completado con exito, aunque faltan datos sobre precipitaciones.
